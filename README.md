# Solar PV Fault Detection using Machine Learning
This project focuses on detecting faults in Solar Photovoltaic (PV) panels using data analysis, SQL, machine learning, and interactive dashboards. The goal is to analyze solar panel operational data and automatically identify faults that can reduce system efficiency. The project follows a complete end-to-end data science workflow including data exploration, SQL analysis, machine learning model development, web application deployment using Streamlit, and visualization using Power BI.

## Project Workflow
Data Collection
        ↓
Data Analysis (Python)
        ↓
SQL Data Analysis
        ↓
Data Preprocessing
        ↓
Machine Learning Model (Random Forest)
        ↓
Model Evaluation
        ↓
Streamlit Web Application
        ↓
Power BI Dashboard

## Dataset
### The dataset contains operational parameters of solar PV panels.

### Features
Feature	                 Description
Irradiance	             Solar radiation received by panel
Avg_Temperature	         Average panel temperature
Humidity_%	             Environmental humidity
PV_Current_(A)	         Current generated by PV panel
AC_Voltage_(V)	         AC voltage output
AC_Current_(A)	         AC current output
AC_Power_(W)	           AC power output
DC_power_(W)	           DC power generated
Efficiency	             Panel efficiency
avg_voltage	             Average voltage
Panel_Class	             Panel condition (target variable)
